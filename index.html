Root Checker

Android application available on the Google Play Store: Root Checker

Root Checker is yet another root checker, however, without asking any dangerous permissions and showing in one screen only, the relevant information necessary for rooting.

Overall, the Root Checker application verifies if the device is rooted and if it has busybox installed by searching for the appropriate binaries in the device. The application also displays device information relevant for rooting, such as product and bootloader codes. Additionally, the application allows the sharing of the verification results.
